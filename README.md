# 🕵️‍♂️ Investing.com USD/KRW Comment Crawler

투자 커뮤니티 포털 [Investing.com](https://kr.investing.com)에서 (타사이트 가능)  
💬  **댓글 데이터를 수집**하는 Node.js 기반 크롤러입니다.

## 📌 주요 기능

- Puppeteer로 페이지 로딩 및 댓글 수집
- 모든 댓글(nth-child 기반) 추출
- JSON 및 CSV 형식으로 저장 가능
- 닉네임, 작성일, 댓글 내용 수집
